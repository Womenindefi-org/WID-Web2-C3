 # GlowCraft by Phina

## Project Description
A responsive landing page for GlowCraft by Phina — a small business that sells curated clothing, handpoured resin art and nourishing lip care products. The page is clean, modern and easy to navigate on any device.

---

## Features
- Responsive design that works on mobile, tablet and desktop
- Navigation bar with logo, links and an Order Now button
- Hero section with a headline and shop button
- Product cards for Clothing, Lip Care and Resin Art
- About section with brand story
- Collections section
- Contact section with WhatsApp and Instagram buttons
- Footer with logo and links

---

## Technologies Used
- HTML5
- Tailwind CSS (via CDN)

---

## Installation Instructions
1. Download or clone this project folder
2. Open the folder in VS Code
3. Make sure your logo and product photos are in the same folder as `index.html`
4. Right-click on `index.html` and click **Open with Live Server**
5. The website will open in your browser

> No installs or packages needed — everything runs in the browser.
