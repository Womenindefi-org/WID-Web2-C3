## 🧩 Challenges Faced

One of the main challenges I faced was setting up the navigation bar.

I initially had issues making it responsive, but I resolved it by using Tailwind CSS grid:

- `grid-cols-1` for mobile screens  
- `grid-cols-3` for desktop screens  

This helped me achieve a responsive layout without using JavaScript for dropdown functionality.
