DevSchool Landing Page
A sleek, minimalist, and fully responsive landing page designed for an educational platform. This project focuses on high-performance UI components, symmetrical design, and utility-first styling.
🚀 Live Demo

🛠️ Tech Stack
• HTML5: Semantic structure for accessibility and SEO.
• Tailwind CSS: Utility-first CSS framework used for all styling, layouts, and animations.
• Lucide Icons / Custom SVGs: Hand-coded SVG paths for a precise, symmetrical user interface.
✨ Key Features
📱 Responsive Mobile Navigation
Implemented a conditional visibility system that swaps standard navigation links for a custom-coded hamburger menu on smaller devices.
• Technical Detail: Uses hidden md:flex and md:hidden breakpoints to manage screen-specific UI elements.
• Symmetry: Features a custom-coded SVG hamburger icon with mathematically equal strokes for a polished look.
🏗️ Advanced Grid Layouts
• Feature Section: A 3-column grid that automatically stacks on mobile for optimal readability.
• Stats Section: A high-contrast "Dark Mode" section showcasing platform achievements with precise spacing and typography.
🎨 Modern UI/UX Elements
• Depth & Interaction: Utilizes hover effects to provide smooth, interactive feedback on buttons.
• Glassmorphism Footer: A sleek, centered footer featuring circular social icons with semi-transparent backgrounds (bg-white/10).

💡 Lessons Learned
Through this project, I deepened my understanding of:
1.	Tailwind Breakpoints: Managing complex layouts across mobile, tablet, and desktop views.
2.	SVG Precision: Writing and debugging SVG path data to ensure icon symmetry.
3.	Deployment Pipelines: Connecting GitHub repositories to Vercel for automated continuous deployment.
