# 🚀 DevSchool Landing Page
A sleek, minimalist, and fully responsive landing page designed for a modern educational platform. This project prioritizes high-performance UI components, symmetrical design, and utility-first styling to create a seamless user experience.

## ✨ Key Features
### 📱 Responsive Mobile Navigation
Implemented a conditional visibility system that swaps standard navigation links for a custom-coded hamburger menu on smaller devices.
 * **Technical Detail:** Uses hidden, md:flex, and md:hidden breakpoints to manage screen-specific UI elements.
 * **Symmetry:** Features a custom SVG hamburger icon with mathematically equal strokes for a polished look.
### 🏗️ Advanced Grid Layouts
 * **Feature Section:** A 3-column grid that automatically stacks on mobile for optimal readability.
 * **Stats Section:** A high-contrast "Dark Mode" section showcasing platform achievements with precise spacing and typography.
### 🎨 Modern UI/UX Elements
 * **Depth & Interaction:** Utilizes smooth hover effects to provide interactive feedback on buttons.
 * **Glassmorphism Footer:** A sleek, centered footer featuring circular social icons with semi-transparent backgrounds (bg-white/10).
