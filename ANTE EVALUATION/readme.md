**Synthos — Web & AI Consultancy Landing Page**
A responsive landing page for Synthos, a fictional Web Design and AI Consultancy agency. Built as part of the WomenInDeFi Web2 Cohort 3 Assessment.
🌐 Live Demo: https://endearing-kataifi-c6f086.netlify.app/

📌 **Project Overview**
This project is a fully responsive, single-file landing page designed for a modern web and AI consultancy agency. It showcases the agency's services, working process, client testimonials, and contact information, all in a clean, professional dark-themed design.

🗂️ **Sections**
Section Description Navigation
Fixed top nav with logo, links, and a CTA button
**Hero Headline, description, two CTA buttons, and key stats**

**Services**
4 service cards — Web Dev, AI Integration, Consulting, Maintenance Process 5-step numbered workflow from Discovery to Growth

**Testimonials**
3 client quotes with names and roles

**CTA Band**
Call-to-action strip with email link

**Footer**
Brand info, navigation links, contact details, and social icons

🛠️ **Tech Stack**

HTML5 — semantic structure and markup
Native CSS — all styling written in a single <style> tag, no external frameworks
Google Fonts — Playfair Display (headings) + Outfit (body text)

📱 **Responsiveness**
The page is fully responsive and tested across:

✅ Mobile (320px and above)
✅ Tablet (768px and above)
✅ Desktop (1024px and above)

Responsive behaviour is handled using CSS @media queries and CSS Grid with auto-fit and minmax().

🎨 **Design Decisions**

Dark theme — professional dark background (#0c0c0c) suited for a tech agency
Gold accent (#c9a84c) — used for headings, labels, and interactive elements to convey premium quality
Teal accent (#2dd4bf) — used to highlight AI-related elements
Playfair Display — an editorial serif font chosen for a refined, high-end feel
Outfit — a clean geometric sans-serif for body text readability
Hover states — subtle color and border transitions on all interactive cards and buttons
